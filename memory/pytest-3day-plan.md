# pytest 3 天学习计划

**时间：** 2026-06-21 起，每天上午

## DAY1：pytest 基础
- 断言 assert
- pytest 运行规则（test_ 开头、函数、类）
- fixture 基础（setup/teardown）

## DAY2：Mock 与 API 测试
- unittest.mock / monkeypatch
- FastAPI TestClient
- 模拟 Redis / MySQL

## DAY3：你的项目实战
- 给你的博客项目写测试
- 测试 login / articles / rate_limit
- 覆盖率报告

---

**为什么先学 pytest？** 因为你现在项目越来越大了，改了代码不敢确定有没有破坏旧功能。有测试就敢重构。
